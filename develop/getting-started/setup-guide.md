<h3>Overview</h3>
The first step to building channels on the Roku platform is setting up a development environment. In this guide, we will cover the essential steps setting up “developer mode” using any Roku device.

<!--more-->
<h4><strong>Steps:</strong></h4>
<ol>
    <li>Setup your Roku Device in <b>“Developer Mode”</b></li>
    <li>Accessing the Development Application Installer</li>
</ol>

<hr />

<h2>Requirements to follow this guide:</h2>
<ul>
    <li>A Roku device: <a href="https://www.roku.com/products/compare">roku.com/products/compare</a></li>
    <li>A Roku account: <a href="https://my.roku.com/signup">my.roku.com/signup</a></li>
    <li>Enroll in the Dev Program <a href="https://developer.roku.com/home">developer.roku.com/home</a></li>
</ul>
<h2>1. Setup your Roku Device in “<b>Developer Mode”</b></h2>
<b>Using your Roku remote (</b><b><i>or Roku iPhone Remote App), enter the following sequence:</i></b>

<a href="https://blog.roku.com/developer/files/2016/02/image05-2.png" rel="attachment wp-att-907"><img class="alignnone size-full wp-image-907" src="https://blog.roku.com/developer/files/2016/02/image05-2.png" alt="image05" width="668" height="90" /></a>

&nbsp;

<b>Note: </b>This secret code enables the Developer Application Installer, which allows you to test channel applications directly on your Roku device.

<b>Follow the steps provided in the following Developer Settings sign up flow:</b>

<a href="https://blog.roku.com/developer/files/2016/02/image00-1.png" rel="attachment wp-att-898"><img class="alignnone size-full wp-image-898" src="https://blog.roku.com/developer/files/2016/02/image00-1.png" alt="Developer Settings" width="1207" height="670" /></a>

Once you begin this process, the dialog will prompt you to restart to continue to the process.

<b>Note: </b>Make sure the save the Roku device URL!

<b>After enabling the installer, review and complete the SDK License Agreement:
</b>You can review the full text at <a href="https://docs.roku.com/doc/developersdk/en-us">docs.roku.com/doc/developersdk/en-us</a>

<a href="https://blog.roku.com/developer/files/2016/02/image03.png" rel="attachment wp-att-901"><img class="alignnone size-full wp-image-901" src="https://blog.roku.com/developer/files/2016/02/image03.png" alt="Read agreement" width="1209" height="676" /></a>

<b>The following prompt will ask to set a password for the Roku device:
</b>This ensures your device is protected on your local area network.

<a href="https://blog.roku.com/developer/files/2016/02/image01.png" rel="attachment wp-att-899"><img class="alignnone size-full wp-image-899" src="https://blog.roku.com/developer/files/2016/02/image01.png" alt="enter dev mode password" width="1208" height="678" /></a>

This completes the steps for enabling Developer mode, continue to the next section for loading applications onto the device.
<h2>2. Accessing the Development Application Installer</h2>
<b>After reboot, open a web browser and enter the Roku device URL </b><i>(i.e. http://192.168.x.x</i>)

<a href="https://blog.roku.com/developer/files/2016/02/image11-e1454565719777.png" rel="attachment wp-att-882"><img class="alignnone size-full wp-image-882" src="https://blog.roku.com/developer/files/2016/02/image11-e1454565719777.png" alt="Logging into Roku Device" width="757" height="382" /></a>

<b>Once the page successfully opens, you have successfully connected to your Roku device in ‘Developer Mode’.
</b>The primary screen will show the development application installer:

<a href="https://blog.roku.com/developer/files/2016/02/image08.png" rel="attachment wp-att-879"><img class="alignnone size-full wp-image-879" src="https://blog.roku.com/developer/files/2016/02/image08.png" alt="Development Application Installer" width="1099" height="525" /></a>
